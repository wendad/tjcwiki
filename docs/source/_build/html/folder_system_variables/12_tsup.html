  • dim-当前背光亮度值(0-100)

  • baud-当前波特率值

  • spax-字符显示横向间距(上电默认为0)

  • thdra-触摸绘图功能开关
  • ussp-无串口数据自动睡眠时间
  • thsp-无触摸操作自动睡眠时间
  • thup-睡眠模式下触摸自动唤醒开关
  • usup-睡眠模式下串口数据自动唤醒开关(上电默认0)
  • wup-睡眠唤醒后刷新页面设置
  • sleep-睡眠
  • lowpower-睡眠模式设定（上电默认为0）
  • bkcmd-设置串口指令执行成功或者失败的数据返回
  • sendxy-实时发送触摸坐标功能开关
  • delay-延时
  • rand-随机数
  • tch0~tch3-实时触摸坐标
  • addr-设备地址
  • crcval-crc校验结果
  • rtc0~rtc6-RTC时钟变量
  • pio0~pio7-扩展IO端口
  • pwm4~pwm7-扩展IO占空比
  • pwmf-PWM输出的频率
