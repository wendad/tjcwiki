cirs-画实心圆

cirs x,y,r,color

x:圆心坐标x

y:圆心坐标y

r:半径

color:填充颜色；

实例1:cirs 100,100,30,RED 以坐标(100,100)为圆心画一个半径为30的实心圆，填充颜色为RED

备注：

关于颜色值的说明请参看cls指令的备注。
