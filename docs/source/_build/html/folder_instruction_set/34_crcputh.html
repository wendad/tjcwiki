crcputh-crc校验一组Hex

crcputh Hex

Hex:需要校验的数据的16进制字符串表达式(每个字节之间用空格隔开)

实例1:crcputh 03 25 (CRC校验hex:0x03,0x25)

备注：

1.使用crcputs或crcputh或crcputu校验指定数据,检验完毕读取系统变量crcval获得校验结果。

2.完整的CRC校验实例代码请参考:程序中使用CRC校验数据
