pic-刷图指令

pic x,y,picid

x:起始点x坐标；

y:起始点y坐标；

picid:图片ID；

实例1:pic 10,20,0 (在坐标(10,20)位置显示资源文件中图片ID为0的图片)

实例2:pic 40,50,1 (在坐标(40,50)位置显示资源文件中图片ID为1的图片)
