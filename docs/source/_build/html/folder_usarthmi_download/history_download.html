指令：crcrest 复位crc初始值(复位之后，可使用crcputs或crcputh或crcputu校验指定数据,检验完毕读取系统变量crcval获得校验结果)

指令：crcputs crc校验一个变量/常量

指令：crcputh crc校验一组Hex

指令：crcputu crc校验一段串口缓冲区数据(recmod=1时有效)

5.版本定义规则从原来的两位版本号更新为3位版本号,分别表示:主版本.子版本.修订号。如：1.60.8 【主版本1，子版本60，修订号8】

主版本变更：一般表示底层做了重大修改

子版本变更：一般代表有新增功能(如新增控件或指令)

修订号变更：一般代表bug修复,没有任何功能变化或底层改动
