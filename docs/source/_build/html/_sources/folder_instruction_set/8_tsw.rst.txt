tsw-控件触摸使能
===============================================================

tsw obj,state

obj:控件名称或控件ID

state:状态(0或1)

实例1:tsw b0,0 (让名称为b0的控件触摸失效) 实例2:tsw b0,1 (让名称为b0的控件触摸有效)

实例3:tsw 1,0 (让ID为1的控件触摸失效) 实例4:tsw 1,1 (让ID为1的控件触摸有效)

备注：

第一个参数 为255表示 当前页面所有控件，例:tsw 255,0(当前页面所有控件触摸失效) tsw 255,1(当前页面所有控件触摸有效)。
