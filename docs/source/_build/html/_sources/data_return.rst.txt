数据返回格式
===============================================================




bkcmd非0时的通知格式
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1.只有当系统变量bkcmd为非0的时候才会返回指令执行成功或者失败数据，每次上电后bmcmd默认为2，即只返回指令执行出错的结果。

2.上位软件编辑时写进资源文件的代码不受bkcmd影响，执行有错误时一定会返回错误数据，成功时不返回数据。

.. toctree::
   :maxdepth: 1

   folder_data_return/return00FFFFFF
   folder_data_return/return01FFFFFF
   folder_data_return/return02FFFFFF
   folder_data_return/return03FFFFFF
   folder_data_return/return04FFFFFF
   folder_data_return/return05FFFFFF
   folder_data_return/return06FFFFFF
   folder_data_return/return09FFFFFF
   folder_data_return/return11FFFFFF
   folder_data_return/return12FFFFFF
   folder_data_return/return1AFFFFFF
   folder_data_return/return1BFFFFFF
   folder_data_return/return1CFFFFFF
   folder_data_return/return1DFFFFFF
   folder_data_return/return1EFFFFFF
   folder_data_return/return1FFFFFFF
   folder_data_return/return20FFFFFF
   folder_data_return/return23FFFFFF
   folder_data_return/return24FFFFFF



其他数据返回格式
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1.设备返回数据的结束符为”0XFF 0XFF 0XFF”三个字节。

2.以下数据的返回不受bkcmd影响。
