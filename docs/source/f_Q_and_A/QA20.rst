20.滑块或进度条控件默认是横向，为什么我设为竖向后没有变化，还是显示横向呢 
========================================================================================================================

默认横向，显示是正常的，如下图：

.. image:: ../media/QandA/1_11.png

但是修改为竖向以后，控件还是横的，如下图：

.. image:: ../media/QandA/1_12.png

这是因为设置为竖向后，软件并不会修改控件的宽度和高度，需要你手动调整他的宽度和高度，调整之后再看显示就是正常的了，如下图：

.. image:: ../media/QandA/1_13.png
